Accessibilité

Ce site a été conçu pour être accessible. Il respecte les critères d'accessibilité du Référentiel Général d'Accessibilité pour les Administrations (RGAA) niveau AA.

Navigation dans le site

La navigation sur le site se fait au travers de la touche Tabulation :
- Un appui sur Tabulation permet de sélectionner le lien suivant sur la page.
- Un appui sur Majuscule + Tabulation permet de sélectionner le lien précédent sur la page.

Une fois un lien sélectionné, il suffit de presser la touche Entrée pour l’ouvrir.

À l’arrivée sur une page, un appui sur la touche Tabulation affiche un menu en haut de page qui permet d’accéder directement à une zone de la page :
- la navigation
- le contenu
- le champ de recherche

Les touches Flèche Haut / Flèche Bas et Page Précédente / Page Suivante permettent de monter ou descendre sur la page.

Grossissement / Diminution de la taille du texte

La taille de texte sur les articles peut être augmentée ou diminuée simplement en cliquant sur les boutons dédiés.
